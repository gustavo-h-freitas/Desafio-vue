SIG VENDAS--------------------------------------O projeto Sig Vendas é o resultado do Desafio Vue.Js com implementação sobre o framework Quasar sobre o NodeJS.O Sistema está integrado com um BaaS chamado Firebase para forncer alguns serviços, como o de banco de dados e autenticação. --------------------------------------FUNÇÕES IMPLEMENTADAS:* Cadastro, edição, exclusão e listagem de produtos* Cadastro, edição, exclusão e listagem de vendedores* Login com o CPF do vendedor.OBS.: As demais funções ainda não foram finalizadas.--------------------------------------PARA EXECUTAR O PROJETO:Após fazer o clone do repositório, é necessário fazer o download das dependências do projeto, para isso usar o seguinte comando:-------------npm i-------------Após fazer a instalação das dependências e entrar na pasta do projeto em um terminal de acesso, use o comando:-------------quasar dev-------------O sistema irá abrir automaticamente uma o seu navegador padrão em uma aba com o seguinte endereço: localhost:8080É nesse endereço que poderá acessar a aplicação em execução em uma instância de desenvolvimento. Caso o navegador não abra ou o link de iniciamento não funcione, por gentileza, colar o seguinte diretamente na barra de navegação do navegador: localhost:8080Na base de dados, um usuário pré-cadastado se encontra disponível para fazer o acesso inicíal e com a autoridade de criar as demais contas e produtos.Utilize para o primeiro acesso o CPF: 123
